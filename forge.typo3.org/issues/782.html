#1 Updated by Steffen Kamper about 7 years ago

here is my proposal

a) darker background for the IRRE-childs to have it clear as different section
b) make complete section collapsable and shows count of childs

darker background: could be any different background

as olly mentioned in bugtracker http://bugs.typo3.org/view.php?id=8267

To be consequent with the collapsable section/icon, also the header of each child record would require such icon. Also the count of items is a nice idea. However there are some (technical) side-effects:

  • plural of object, e.g. "prices", "offers" is not possible or you have to define the plural for each object name - adding only an "s" for English Plural would not work in all cases - in German its more difficult, e.g. for "Preise" or "Hotels" -> so we just use "items" or "item"
  • each dynamic action (e.g. localizing/synchronizing) records has to take care about this correct count
  • notification of unfilled required fields has to be taken into account for the collapsable section

I think there is a lot of potential in it.

  • the plural problem could be simplified with using record/records
  • record count can be complex, but could be a tooltip, where you show complete number, and on hover there is details shown like
    flag default - 12
    flag de - 8
    etc.
  • notification can be done with the notification-icon, should be no problem to integrate it in the header

and yes, i would prefer the +/- icon for each header (groups and elements)
