#3 Updated by Philipp Gampe over 1 year ago

You need to differentiate TCEFORM.tt_content.colPos.removeItems = 1,2,-2 which removes certain columns for a usergroup or below a certain page and mod.SHARED.colPos_list = 0,1,2,3,-4 as well as mod.web_layout.tt_content.colPos_list = 0,3 which change the visual appearance of the of page module (and the available items list).

The former still works. The latter is deprecated, because the same can be archived with backend layouts.
