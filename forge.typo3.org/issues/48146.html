Work Package #45020: Content Editing Stability and Features

Change default generated node paths when changing the node title

Added by Aske Ertmann about 2 years ago. Updated 9 months ago.

Status:Closed Start date:2013-05-13
Priority:Should have Due date:
Assigned To:Aske Ertmann % Done:

80%
Category:- Spent time: -
Target version:1.0 alpha 6

Description

A default node path is generated when creating a new node, we should change this automatically when a node with a title property is changed if it still has the default node path. This is to make it easier for the editor to get nice URL's. For now we should however not change this automatically if the title changes and the node path is not a generated one, because we lack a good way to support redirects from old URL's.
