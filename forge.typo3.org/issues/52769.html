Document advanced metadata for File

Added by Fabien Udriot almost 2 years ago. Updated almost 2 years ago.

Status:New Start date:2013-10-14
Priority:Should have Due date:
Assigned To:- % Done:

0%
Category:-
Target version:-

Description

The Core provides advanced metadata for File within a Core
extension. The extension is not activated by default but can be
when advanced metadata are needed.

The main benefit is to have unified metadata in TYPO3.
As a result, third-party extensions can build upon a common
base preventing segmentation.
