Implement new layout and labels for tt_content as result of T3UXW09

Added by Jo Hasenau almost 5 years ago. Updated over 4 years ago.

Status:Closed Start date:2010-10-17
Priority:Should have Due date:

Complexity:

Description

This is one of the results of team 1 of the user experience week.

It implements a completely new structure for tabs, fieldsets, fields and labels for the TCE form of tt_content via palettes and types only.

Additionally it introduces so called "formlabels", that enable us to use different and more verbose labels for the forms, while keeping the default labels for the rest of the system intact.

Some of the labels are still hardcoded within the TCA settings, but the look and feel can already be tested.

To test all different elements, uninstall CSS-styled-content to avoid elements being disabled.

German translations will follow.
