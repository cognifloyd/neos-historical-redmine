f:uri.resource do not have parameter of title

Added by Krzysztof Daniel about 3 years ago. Updated about 3 years ago.

Status:Resolved Start date:2012-06-25
Priority:Could have Due date:
Assigned To:Karsten Dambekalns % Done:

100%
Category:- Documentation -
Target version:TYPO3 Flow Base Distribution - 1.1
PHP Version: Complexity:
Has patch:No Affected Flow version:FLOW3 1.1.0 beta 3

Description

http://flow3.typo3.org/documentation/guide/partiii/resourcemanagement.html gives an example:

{f:uri.resource(resource: image.originalResource, title: image.title)}

This seems not to be implemented as this helper do not have parameter of title.

Either helper or documentation shall be corrected.
