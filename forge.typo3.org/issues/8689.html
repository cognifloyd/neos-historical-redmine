#11 Updated by Christopher Hlubek over 4 years ago

I would propose to use the jQuery UI Sortable Widget: http://jqueryui.com/demos/sortable/#items. The plugin is flexible and can be used to sort just one item (the page to be created) into a list of existing items. I think it shouldn't be a problem to write an Ext JS Component as a wrapper that uses this plugin and can be used inside a form.

We are already using jQuery for Aloha by the way.
