Description

In the cgl there is a doc block comment example

/**
 * Initializes the plugin.
 *
 * Checks the configuration and substitutes defaults for missing values.
 *
 * @param array $conf Plugin configuration from TypoScript
 * @return boolean TRUE if initialization was successful, FALSE otherwise 
 * @see tx_myext_class:anotherFunc()
 */

// Do something
}

It must be discussed if the long types are mandatory or it if free to the developer.
Lets create a ticket at the cgl forge project. After this we can determine if there must be a sniff :)
