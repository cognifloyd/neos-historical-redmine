Update documentation of options.clearCache.clearRTECache

Added by Chris topher over 2 years ago. Updated over 1 year ago.

Priority:Must have Due date:
Assigned To:- % Done:

Description

The documentation of options.clearCache.clearRTECache needs an update and options.clearCache.clearLangCache needs to be added.

It seems like options.clearCache.clearRTECache was removed. Instead this option is now called options.clearCache.clearLangCache. See #39103. (Also see here, slide 28)

Check if the property options.clearCache.clearRTECache is still used at other places or if it really was removed everywhere.

  • mark options.clearCache.clearRTECache as removed or document that its behaviour changed (depending on what it is) in TYPO3 6.1 and
  • add documentation for options.clearCache.clearLangCache
  • add the change to the section "What's new" as it is a breaking change
