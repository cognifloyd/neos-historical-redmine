Please read the notes in story #40587 first.

In order to delete a page using the page tree (Content Editing View):

  • select the page to delete
  • click the trash bin icon in the action toolbar on top of the page tree
  • confirm a "Cancel / Delete" dialog

The Cancel/Delete dialog should be modal. It asks if the page and all sub pages (if any) should be deleted and mention the page title.
