#3 Updated by Jens Hoffmann over 4 years ago

Via Skype we decided (for this first release) to keep it like it is right now.

For the future we would wish to have:

4 ways of comparison:

  • Horizontal overlay (via Horizontal slider) << like it is right now
  • Vertical overlay (via Horizontal slider)
  • Opacity overlay (via Horizontal slider)
  • Horizontal Split view 50% Top / 50% Bottom

This could be set:

  • Via a context Menu beside the Slider UI
  • Via User Settings (TS)
  • Via Global TS for a installation

Would be very helpful if someone who dislike the current approach
would help out and continuing the existing development by creating
the planned roadmap steps, as suggested here. :) Susanne?
